
# Project 07-Muzicka-desavanja

Plan je da napravimo web sajt koji ce korisnicima pruzati informacije i muzickim dogadjajima (npr. u Beogradu), online kupovinu karata(ili samo prijava za dolazak),pregled dogadjaja i termina odrzavanja...Takodje plan je da se napravi pravljenje korisnickih i administratorskih(kao vip akaunti koji mogu da dodaju same muzicke manifestacije) akaunta...Koristili bi Mongo bazu i node server...
